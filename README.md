## My Blog 

>本项目为一个个人的网站的后台程序的模板（2021年2月13日进行开发开发中）
> - 支持2次开发
>   - app/pc 端 后台服务端api接口开发
>   - 后期自定义admin界面api服务
### 基本功能

- 用户模块
  - 用户登录
  - 用户注册
  - 用户重置密码
  - 修改用户信息
  - 用户上传头像
  - 找回密码
  - 得到手机验证码(请绑定手机验证码第三方服务接口)
  - 手机验证码验证
  - 用户信息显示（全部/个人）
  - 注销用户
    
- 自动生成项目的api 接口文档
    - swagger 风格的api接口文档
    - redoc 风格的api接口文档
    - drf 风格的api接口文档
